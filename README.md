# Predictive Modeling Using Machine Learning

## Project Overview

This project demonstrates the complete machine learning workflow using the Titanic dataset. The objective is to predict passenger survival using Logistic Regression after performing data preprocessing and exploratory data analysis.

---

## Dataset

- Titanic Dataset
- Target Variable: Survived

---

## Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Workflow

- Data Loading
- Data Cleaning
- Handling Missing Values
- Feature Encoding
- Exploratory Data Analysis (EDA)
- Feature Selection
- Train-Test Split
- Logistic Regression Model
- Model Evaluation

---

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve
- ROC-AUC Score

---

## Model Performance

- Model: Logistic Regression
- Accuracy: **81.01%**

## Conclusion

The Logistic Regression model achieved an accuracy of **81.01%**, demonstrating good predictive performance on the Titanic dataset. This project showcases essential machine learning concepts including data preprocessing, visualization, model training, and evaluation.
